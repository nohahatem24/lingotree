LingoNest is a comprehensive e-learning platform built to provide engaging English courses for children. Developed using AI-assisted tools like **Cider Web App Builder**, the platform was then heavily customized to create a unique, branded, and feature-rich user experience. It allows parents to sign up, purchase courses, and monitor their children's progress, while kids can enjoy interactive quizzes, earn badges, and communicate directly with their teachers in a safe, moderated environment.

## ğŸš€ Project Vision & My Role

As the **Lead Product Designer and UI/UX Specialist**, my goal was to transform an AI-generated foundation into a bespoke, human-centered platform. My passion was to create an environment that felt both playful for children and trustworthy for parents.

My core responsibilities included:
*   **End-to-End Design Leadership:** I led the entire design process, taking the initial concept from Cider and reimagining it into a fully customized and branded platform that met specific user needs.
*   **User-Centric UX for All Roles:** I designed the complete user experience (UX) and user interface (UI) for all three user roles, ensuring that the journey for a child, a parent, and a teacher was intuitive, distinct, and effective.
*   **Gamified & Engaging UI for Kids:** I focused heavily on creating a visually engaging and gamified learning environment for the primary usersâ€”children. This involved designing the badge system, selecting a vibrant color palette, and creating interactive elements that make learning feel like play.
*   **Data-Driven Dashboards:** I designed the parent and teacher dashboards to be clean, intuitive, and data-rich, providing clear, actionable insights into student progress and activities.
